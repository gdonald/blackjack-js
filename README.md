# blackjack-js

Blackjack in ReactJS (Typescript)

#### Install

``
npm install
``

#### Run

``
npm start
``

#### Play

[http://localhost:8080/](http://localhost:8080/)

#### Demo

[https://games.gregdonald.com/blackjack/](https://games.gregdonald.com/blackjack/)
